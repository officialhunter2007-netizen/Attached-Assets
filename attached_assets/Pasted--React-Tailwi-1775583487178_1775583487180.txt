القسم الذكي (المميز الأساسي):
  - يتحقق هل للمستخدم مسار محفوظ في UserLearningPath
  - إذا لا: يعرض بطاقة دعوة لبدء الجلسة الذكية

  - الضغط يفتح مكوّن SubjectPathChat

SubjectPathChat — ثلاث مراحل:

  المرحلة ١ (INTERVIEWING):
  - واجهة دردشة نصية
  - Claude يسأل 3-5 أسئلة بالتسلسل (مستوى، هدف، وقت، نقاط ضعف)
  - بعد كل إجابة يسأل Claude: هل لدي معلومات كافية؟
  - إذا نعم: يرد بـ READY فقط (startsWith check)
  - إذا لا: يسأل سؤالاً إضافياً
  - الحد الأدنى 3 أسئلة قبل التحقق

  المرحلة ٢ (BUILDING → PLAN_SHOWN):
  - يولد Claude خطة HTML احترافية كاملة
  - تُعرض في iframe بخلفية داكنة وتصميم فاخر
  - تشمل: تحليل المستوى، مراحل التعلم، جداول أسبوعية، موارد، الهدف النهائي
  - تُحفظ في UserLearningPath
  - أسفلها: زر "ابدأ الجلسة التعليمية"

  المرحلة ٣ (TEACHING):
  - واجهة دردشة تعليمية حية بارتفاع ثابت 680px
  - رسائل المعلم: iframe مع CSS كامل مدمج، تتمدد تلقائياً
  - رسائل الطالب: فقاعة نصية ذهبية
  - CSS المدمج في كل iframe:
    * خلفية شفافة، خط Tajawal/Cairo
    * h3 ذهبي، h4 زمردي، strong أصفر، em أخضر
    * pre>code: خلفية #0d1117، لون أزرق، LTR
    * code مضمّن: خلفية ذهبية خفيفة
    * .question-box: بوردر ذهبي للسؤال النهائي
    * .tip-box: بوردر زمردي للتلميحات
    * .praise: لون زمردي للإطراء
  - آلية التدريس:
    * مفهوم واحد فقط في كل رسالة
    * مثال عملي قابل للتطبيق الفوري
    * سؤال/تحدٍّ في نهاية كل رد داخل .question-box
    * تصحيح بلطف + مديح بإخلاص + انتقال للتالي
    * كل شيء HTML داخل <div> — لا Markdown

أسفل الجلسة الذكية:
  - Accordion يعرض جميع الوحدات والدروس (للتنقل المباشر)

قواعد الكود العامة
═══════════════════════════════════════════════════════

- كل component في ملف منفصل
- الملفات الكبيرة (+150 سطر) تُقسَّم لـ sub-components
- لا أكواد وهمية أو TODO — كل شيء مكتمل وقابل للتشغيل
- لا Markdown داخل رسائل Claude التعليمية — HTML فقط
- RTL في كل مكان إلا الكود البرمجي
- جميع الألوان من متغيرات CSS أو utility classes — لا hex مباشر
- responsive في كل الأجهزة (mobile-first)
- framer-motion لكل الحركات (initial/animate/exit)
- لا console.log في الكود النهائي
هذا البرومبت يغطي كل تفصيلة في المنصة — من الثيم والـ entities وصولاً لكل صفحة وكل component وكل interaction وكل برومبت AI داخلي. يمكنك إعطاءه لأي AI مطور وسيبني نسخة مطابقة للمنصة.

📋 برومبت شامل لمنصة نُخبة التعليمية
🎯 البرومبت الكامل (انسخه كما هو)
أنت مطور ويب خبير متخصص في React + Tailwind CSS + Base44 SDK.
أريدك أن تبني منصة تعليمية يمنية متكاملة اسمها "نُخبة" من الصفر.
